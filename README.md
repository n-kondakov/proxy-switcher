# 🌐 Proxy Switcher

Удобное управление прокси для браузеров на базе Chromium(Chrome, Edge, Yandex): глобальный режим, правила по доменам, авто-переключение при сбоях, защита от утечек и разблокировка 1080p/1440p на Twitch.

## ✨ Возможности

#### 🔧 Управление прокси
- Поддержка **SOCKS5** и **HTTP** прокси
- Глобальный режим: весь трафик через выбранный сервер
- Точечные правила: прокси только для указанных доменов
- Быстрое редактирование, удаление и переключение

#### 🔄 Авто-свитчер (Failover)
- Отметь прокси как `AUTO` и расширение автоматически переключится на запасной, если текущий перестанет отвечать

#### 🛡️ Защита приватности
- Блокировка утечек WebRTC - реальный IP не раскрывается

#### 🎮 Twitch 1080p / 1440p
- Обход региональных ограничений качества на Twitch (для РФ и других регионов)
- Запросы к `usher.ttvnw.net` перенаправляются через прокси-серверы `rte.net.ru`
- Видео загружается напрямую с CDN - твой основной прокси не меняется

#### 💾 Импорт / Экспорт
- Сохрани все настройки в `.json` и восстанови на другом устройстве за секунду

## 🚀 Установка

1. Скачай или склонируй репозиторий:
   ```bash
   git clone https://github.com/undecide/proxy-switcher.git
2. Открой браузер и перейди в `chrome://extensions/`
3. Включи **Режим разработчика**
4. Нажми **Загрузить распакованное расширение** и выбери папку с проектом

## ⚙️ Разрешения и почему они нужны

| Разрешение | Зачем |
|------------|-------|
| `proxy` | Управление системными настройками прокси |
| `declarativeNetRequest` | Перенаправление запросов Twitch |
| `privacy` | Контроль WebRTC-утечек |
| `storage` | Сохранение настроек и списка прокси |
| `webRequest` / `webNavigation` | Мониторинг ошибок прокси |
| `alarms` | Периодическая проверка статуса |
| `tabs`, `cookies` | Для будущей интеграции с авторизацией |
